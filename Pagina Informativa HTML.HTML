PAGINA INFORMATIVA DE HTML

HTML (HyperText Markup Language) es el lenguaje estándar para la creación de páginas web. Es un lenguaje de marcado que estructura el contenido mediante etiquetas, definiendo elementos como texto, imágenes, enlaces y otros componentes.

PRINCIPIOS BASICOS DE HTML

Estructura de Documentos HTML: Un documento HTML comienza con la declaración !DOCTYPE html, seguida de las etiquetas html, que encierran el contenido de la página.

Se divide en dos partes:

  1. head: Contiene metadatos, títulos, enlaces a CSS y scripts.
  2. body: Contiene el contenido visible, como párrafos, imágenes y enlaces.

ETIQUETAS

HTML utiliza etiquetas para definir la estructura del contenido:

  1. h1 - h6: Encabezados.
  2. p: Párrafo.
  3. a: Enlace.
  4. img: Imagen.
  5. div y span: Contenedores para agrupar elementos.

ATRIBUTOS

Las etiquetas pueden tener atributos que definen propiedades adicionales.

MULTIMEDIA E INTERACTIVIDAD

HTML admite elementos multimedia como:

  1. img: para imágenes.
  2. video y audio: para video y audio.

CARACTERISTICAS CLAVE DE HTML

  1. Lenguaje de marcado: HTML organiza el contenido de manera estructurada usando etiquetas.
  2. Independencia de plataforma: Funciona en todos los navegadores y dispositivos.
  3. Extensibilidad: HTML puede integrarse con CSS (estilos) y JavaScript (interactividad).
