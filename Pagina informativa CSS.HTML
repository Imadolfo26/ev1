PAGINA INFORMATIVA CSS

ELEMENTOS Y PROPIEDADES DE CSS

CSS es un lenguaje utilizado para describir el aspecto y el formato de un documento HTML. CSS define cómo se presentan los elementos en la página, tales como colores, tamaños, fuentes, alineaciones, y mucho más.

ELEMENTOS

  • SELECTORES

    Los selectores son patrones utilizados para seleccionar los elementos HTML a los que se aplicará el estilo.

    • Selector de etiqueta (element): Aplica estilos a todos los elementos de un tipo específico.
    • Selector de clase (.class): Aplica estilos a todos los elementos con una clase específica.
    • Selector de ID (#id): Aplica estilos a un único elemento que tiene un ID específico.
    • Selectores universales (*): Aplica estilos a todos los elementos de la página.
    • Selectores de atributos: Aplica estilos a los elementos que tienen un atributo específico.
  • PROPIEDADES

    Las propiedades de CSS definen las características visuales de los elementos seleccionados. Cada propiedad tiene un valor que determina cómo se aplicará el estilo.

  • VALORES

    Los valores definen cómo se comporta o aparece la propiedad de un elemento.

  • REGLAS

    Una regla de CSS consiste en un selector y un bloque de declaración. El bloque de declaración contiene una o más declaraciones CSS, cada una con una propiedad y un valor.

PROPIEDADES COMUNES DE CSS

PROPIEDADES DE TEXTO Y FUENTES
  1. color: Color del texto.
  2. font-family: Tipo de letra.
  3. font-size: Tamaño de la fuente.
  4. text-align: Alineación del texto (izquierda, derecha, centro).
FONDOS
  1. background-color: Color de fondo.
  2. background-image: Imagen de fondo.
  3. background-size: Tamaño de la imagen de fondo.
MODELO DE CAJA
  1. width y height: Ancho y alto de un elemento.
  2. padding: Espacio interno.
  3. margin: Espacio externo.
  4. border: Borde del elemento.
POSICIONAMIENTO
  1. position: Posición de un elemento (static, absolute, relative).
  2. z-index: Superposición de elementos.
